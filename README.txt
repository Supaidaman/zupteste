---------COMO USAR---------

Existem duas maneiras básicas de usar este programa, no Windows:

1) Abrindo o projeto pelo Eclipse. 
  1.1) Abrindo o Eclipse, vá em File->Import->General->Existing Projects Into Workspace
  1.2)Em Select Root Directory, selecione a pasta onde você clonou (ou descompactou) este repositório.
  1.3) Selecione o projeto na lista e clique em Next.
 1.4)Com o projeto aberto, clique em run(executar) e então coloque as informações pertinentes na linha de comando da IDE. O resultado ira aparecer abaixo do seu input nesta propria linha de comando.

2) Através do JAR. (método prefírivel)
  2.1) Para executar o executavel.jar, coloque-o junto do arquivo testefile.txt(onde ficam as instruções pro programa)
  2.2) Abra o cmd(linha de comando) e navegue para a pasta onde o .jar está com o comando "cd"
  2.3) Ao chegar na pasta, execute java -jar executavel.jar
  2.4) Os resultados aparecem na linha de comando.
  2.5) Para modificar o input, mude os valores dentro da testefile.txt a sua vontade.
